Ejercicio: Tablas y Listas Avanzadas

En este ejercicio deberás crear tablas y listas complejas que combinen diferentes atributos y estructuras. Sigue las indicaciones a continuación:

Parte 1: Listas

Crea una sección de listas complejas siguiendo los siguientes pasos:

  • Crea una lista desordenada con tres categorías principales (por ejemplo: "Películas", "Libros", "Series"). Cada categoría debe contener una lista ordenada con al menos tres elementos, enumerando tus favoritos en cada categoría.
  • En una de las categorías, utiliza listas anidadas para incluir subcategorías. Por ejemplo, en "Películas" podrías añadir subcategorías como "Acción", "Comedia" y "Drama", con elementos correspondientes en cada subcategoría.
  • Cambia el estilo de las listas desordenadas (circle, square, disc) y el tipo de las listas ordenadas (números romanos, letras mayúsculas y minúsculas).
  • Utiliza el atributo start en al menos una de las listas ordenadas para que no comience desde el valor predeterminado.

Parte 2: Tablas

Crea una tabla compleja para mostrar un horario semanal ficticio de tus actividades con los siguientes requisitos:

  • La tabla debe incluir un encabezado con los días de la semana como columnas y las franjas horarias (Mañana, Tarde, Noche) como filas.
  • En el cuerpo de la tabla, rellena las actividades correspondientes para cada combinación de día y franja horaria.
  • Fusiona celdas verticales o horizontales donde las actividades sean las mismas para una franja horaria o un día completo, utilizando los atributos rowspan o colspan.
  • Incluye un pie de tabla con una celda que abarque todas las columnas, indicando "Horario sujeto a cambios".
  • Añade un borde visible a la tabla y usa cellspacing y cellpading a tu gusto.
